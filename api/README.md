# zetria.org APIs  

**No API keys** are needed. 😉  
APIs include (in alphabetical order):  
  
- **External Websites API**  
 -> *其他教育網站的連結*
- **External Websites Result API**  
 -> *搜尋其他教育網站的資料庫*
- **Report API**  
 -> *回報 學呀 資料庫中的問題資料*
- **Result API**  
 -> *搜尋 學呀 資料庫中的教育資源*
- **Similar Terms API**  
 -> *搜尋近義詞彙*
